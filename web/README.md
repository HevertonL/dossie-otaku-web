# 🎨 Frontend — Dossiê Otaku

Interface web da aplicação, responsável pela experiência do usuário.

---

## 🛠️ Tecnologias

- React
- Vite
- Axios
- Cypress

---

## 🚀 Como rodar

```bash
cd web
npm install
npm run dev
```

---

## 🔐 Variáveis de Ambiente

```env
VITE_API_URL=http://localhost:3000
```

---

## 🧪 Testes E2E

```bash
npx cypress open
```
